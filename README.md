# SpendWise

SpendWise is a simple and user-friendly expense tracker that allows users to manage their finances by adding, editing, and deleting transactions.

## Features
- Add income and expenses
- Edit existing transactions
- Delete transactions
- View total balance
- Persistent data using localStorage

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How It Works
Users can input their income and expenses, which are stored in the browser using localStorage. The application dynamically updates the total balance and displays all transactions in a list.

## Project Status
This project is currently under active development. New features and improvements are being added regularly.

## Purpose
This project was built to practice JavaScript fundamentals and demonstrate CRUD (Create, Read, Update, Delete) functionality in a real-world application.

## Future Improvements
- Add categories (e.g., food, rent, transport)
- Implement charts for spending insights
- Add user authentication
- Connect to a backend database
